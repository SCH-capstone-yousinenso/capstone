# FITBIT 데이터 추출 방법

## 1. Fitbit 계정 만들기 : https://www.fitbit.com/

## 2. https://dev.fitbit.com/ 들어간 후 상단 Manage -> Register An app 클릭

## 3. 빨간 표시 똑같이 할 것
> Oauth 2.0 Application Type은 personal 체크
> Callback URL:http://127.0.0.1:8080/
> 작성 후 submit
